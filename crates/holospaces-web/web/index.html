Lifecycle

Publishes a lifecycle Configuration (CC-28); the running instance resolves, verifies, and applies it over the control channel (ADR-018).

Network

Forwarded guest ports become app-preview URLs; fetch/announce govern the guest's substrate connectivity. Egress node: a tab has no NIC, so the guest's internet (curl/apt) routes through a holospaces-node you own (CC-39) — the mesh's exit, not a proxy. Leave blank for content-only (no live egress).

No node? Install the local egress extension — it opens raw sockets in your own browser (Direct Sockets, CC-41), so a Chromebook reaches the internet with nothing else. ⬇ Download extension. Manual install (until it's in the Chrome Web Store): download → unzip → chrome://extensions → enable Developer mode → Load unpacked → pick the unzipped folder.
